Book Tracker
Автор: Волкова Амина Рафисовна

Описание программы:
GUI-приложение для учёта прочитанных книг. Позволяет добавлять книги в таблицу, фильтровать их по жанру и количеству страниц, а также сохранять и загружать данные в формате JSON.

Требования при запуске и пользовании:
- Установлен Python 3.6+
- Встроенны библиотеки: `tkinter`, `json`, `os`, `messagebox`

  git clone <URL_РЕПОЗИТОРИЯ>
